Write a function that takes in list containing strings and numbers as an argument and returns one string containing the sum of all the numbers in the list followed by a comma followed by a comma and the rest of the words (all separated by commas). Assume all the numbers in the list are of type int. For example [1, 2, 3, "what", "hello"] would return "6,what,hello"
skill: 9, 12, 30
Description


def list_check(data_list):
    total = 0
    s = ""
    for item in data_list:
        if type(item) == int:
            total = total + item
        elif type(item) == str:
            if s != "":
                s += ","
            s += item
    result = f'{total},{s}'
    
    return result
print (list_check ([1, 2, 3, "what", "hello"]))

Create a function that can take in multiple delimiters in the form of a string, separated by another character that will not be used as a delimiter.
Parameters include:
string with multiple delimeters separated by a given character
the given character
the string
ex:
multiDelimeter("=/__/]", "/", "hello=i__am__a]goat") ==> ["hello", "i", "am", "a", "goat"]
your function must separate them into a single list, without any sublists.


def multiDelimeter(d, sep, listr):
    separators = d.split(sep)
    strr = listr
    Wstr = listr
    for delim in separators:
        Wstr = strr.split(delim)
        #print(f'Wtsr Del:{Wstr}')
        strr = ""
        for i in Wstr:
            strr += i + sep
        strr = strr[:len(strr)-1]
    return strr.split(sep)

    count = 0
    for i in s:

    even_list = []

        if int(e) % 2 == 0:
            even_list.append(int(e))
